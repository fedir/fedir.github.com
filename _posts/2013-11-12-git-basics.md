### git

It helps to control the code, it's changes during time, even if lot's of people working on it.

### Workflow difference betwet git and svn

* SVN
  - Workspace
  - Index
  - Repository
 
* git
  - Stash
  - Workspace
  - Index
  - Local Repository
  - Upstream Repository
